# 各种各样的 Python 小项目

> 用于学习、练习与实验的 Python 小项目合集。

## 仓库说明

这个仓库主要用来存放我在学习 Python 过程中写的一些 **小型练习项目 / 脚本 / 实验代码**。  
项目规模不大，侧重于：
- 熟悉 Python 语法与标准库
- 通过「做点什么」来巩固知识
- 记录学习过程中的想法与尝试

## 内容类型

仓库中的项目大致包括但不限于：

- 🧩 **基础练习**  
  语法、流程控制、数据结构等基础内容的练手代码

- 🛠 **小工具 / 脚本**  
  一次性脚本、简单自动化、小功能工具

- 🎮 **简单项目 / 小游戏**  
  规模较小、逻辑清晰的完整小项目

- 🧠 **算法与思维训练相关代码**  
  用 Python 实现的算法练习或思路验证
